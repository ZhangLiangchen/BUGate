# integration workbench

This directory holds the generated onboarding artifacts produced by the
`integrate-qa-methodology` workflow — the current-scheme inventory, the
methodology requirement list, the integration plan, and the dry-run report. It
is an output directory only: nothing here is part of the BUGate engine or an
install prerequisite, and its generated contents stay out of BUGate core
(gitignored). The canonical workflow body lives in
`../commands/integrate-qa-methodology.md`.
