# Layer 2 Testability Gate

Layer 2 is acceptable only when it explains:

- Which propositions are covered by which test layer.
- Why a cheaper layer is insufficient for any higher-layer case.
- What evidence source will drive each assertion.
- Which dependencies, fixtures, and resources are required from the SUT profile.
- Which claims remain deferred and who must resolve them.

Reject Layer 2 when it jumps directly from requirement prose to end-to-end tests
without a layer decision.
