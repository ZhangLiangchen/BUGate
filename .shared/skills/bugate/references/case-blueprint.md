# Layer 3 Case Blueprint Gate

Layer 3 is acceptable only when each case has:

- A stable case identifier and intent.
- Proposition references.
- Oracle references for business-critical cases.
- Preconditions and data source status.
- Expected observable result.
- Test layer decision and implementation target owned by the SUT profile.

Reject Layer 3 when coverage is implicit, evidence is unnamed, or generated test
code would need to invent missing data.
